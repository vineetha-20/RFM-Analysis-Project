# RFM Analysis Project

This project analyzes customer transaction data using RFM (Recency, Frequency, Monetary) principles to segment customers and understand business value.

## What this project includes
- A Jupyter notebook for data cleaning, transformation, and analysis
- Customer segmentation based on RFM scoring
- Sales forecasting using time-series modeling
- Output files for segmentation and forecasting results

## Files in this repository
- cleaning.ipynb: main notebook for data processing and analysis
- Raw data.csv: original transaction dataset
- rfm_output.csv: customer segmentation results
- historical_daily_sales.csv: daily historical sales data
- sales_forecast_output.csv: forecasted sales output
- customer.pbix: Power BI report file

## How to use
1. Open cleaning.ipynb in Jupyter Notebook or VS Code.
2. Run the cells in order to clean the data and generate the analysis outputs.
3. Review the generated CSV files for the results.

## Requirements
- Python
- pandas
- numpy
- statsmodels
- Jupyter Notebook or VS Code with Python support

## Purpose
The goal of this project is to identify valuable customers, detect churn risk, and support better business decisions through data-driven segmentation and forecasting.
